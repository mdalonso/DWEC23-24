UD3.-Estructuras definidas por el usuario en JavaScript

05-Mapas

Crear un script que utilice la estructura de map para realizar el seguimiento de las puntuaciones de los jugadores. Se pedirá mediante prompt el nombre y la puntuación. El nombre no podrá ser vacío y al pulsar el botón Cancelar, finalizará la entrada de datos (función, devuelven el dato) Las puntuaciones, se controlarán que sean numéricas y no se permitirán datos nulos (función, devuelven el dato) A continuación, se ordenará por puntuación ascendente y se mostrará el mapa ordenado. Utilizar funciones para la entrada de datos, otra para la ordenación y otra para mostrar el mapa ordenado.
